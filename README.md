# MyNote

MyNote is a simple personal Markdown notebook built with Tauri, React, and TypeScript.

## 菜单与右键菜单

- 顶部系统菜单提供全局入口：文件、编辑、视图、笔记、帮助。
- 文件树、标签、编辑器选区和空白区域都提供对象化右键菜单。
- 部分未来能力当前会以灰态菜单项显示，用于固定后续扩展位置。
